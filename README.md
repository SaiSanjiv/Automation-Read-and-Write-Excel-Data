This UiPath project reads data from an Excel file, processes each row, and writes the updated data into a new output Excel file.

## Features

* Read Excel data
* Update or process rows
* Write results to output Excel

## Files

* Main.xaml
* project.json
* input.xlsx (sample)
* output.xlsx (sample)

## How to Run

1. Open the folder in UiPath Studio.
2. Place your input file in the project folder.
3. Run Main.xaml.
4. Updated results will appear in output.xlsx.
